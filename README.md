# Project-Trajectory-Prediction

Voilà le projet de Deep Learning et IA générative & Grands modèles de langage de Timéo Gogolachvili et Rémi Mathivanan.
Il s'agit d'un projet pour comparer l'efficacité de 4 modèles d'IA pour la prédiction de trajectoire de piétons.

Pour l'ouvrir :
- Téléchargez le fichier .ipynb
- Ouvrez un terminal
- Utilisez la commande "jupyter notebook" (installez le via le terminal si pas encore fait)
- Votre navigateur s'ouvre, séléctionnez le fichier
- Ne fermez pas le terminal pendant que vous avez le fichier ouvert !

Précision, pour qu'il fonctionne, vous devez modifier le chemin dans le début du code pour le chemin d'accès de votre fichier dataset (télécharger d'abord le dataset ETH/UCY).
